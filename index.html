About Me

Hello! My name is Miloš Kocić. I have a Bachelor's degree in Computer Science (Prirodno-matematički fakultet - Niš). Responsible, ambitious, well organized, creative, resourceful and hard working IT person with great problem-solving skills. I'm able to work both in a team environment as well as own initiative.

I currently working as 3D Designer (mostly jewelry). I make jewelry pieces like rings, pendants, necklace, earrings from a given sketch/idea/image using Rhinoceros3D and high quality jewelry rendering services using KeyShot (preparing STL files for 3D printing and make renders, animations...).

But, my passion for web programing is something that I really want to do. I gained solid knowledge of object-oriented programming in college and many programing languages like: C, C++, Java, HTML, CSS, JavaScript, Bootstrap, MySQL, PHP, Git. And I know Photoshop very well. But I mostly like to work with frontend technologies and because of that recently start to learning JavaScript framework React.
